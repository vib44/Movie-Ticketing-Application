# Node.js Express Project using MongoDb

## Setting up Project

Create client and server folders.
Under server folder -
    
    1. Install the Node.js dependencies using:
    
        npm init -y

    2. Install all other tools

        npm i mongoose nodemon express    

    3. Write dbConfig.js and check connection by running below command

        ./server>
        nodemon index.js

    4. Install dotenv to use .env file for configs

        ./server>
        npm i dotenv
